# Fase 09 - Reportes ERP y BI

## Objetivo

Profundizar los reportes comerciales y dar visibilidad gerencial real sobre ventas.

## Alcance funcional

- rentabilidad por ventas
- total de margen comercial
- ventas por canal
- top vendedores
- top zonas
- serie diaria de ventas
- saldos pendientes

## Alcance tecnico

- ampliar `salesReportData`
- enriquecer vista web y PDF
- exponer datos agregados por API

## Criterios de aceptacion

- El reporte muestra indicadores comerciales y de margen.
- El PDF incluye los nuevos bloques.
- Los datos son consistentes con ventas confirmadas y cobranzas.
