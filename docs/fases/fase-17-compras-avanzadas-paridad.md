# Fase 17 - Compras avanzadas

## Objetivo
Profundizar Compras para acercarse al comportamiento de Tango Proveedores.

## Alcance implementado
- Facturas de proveedor.
- Notas de credito de proveedor.
- Historial de costos por proveedor/producto.
- Diferencias de cambio en pagos no ARS.
- Integracion de pagos con pasarela, cheque y referencia externa.

## Validacion tecnica
- Migracion de tablas de compras avanzadas.
- Web y API REST para facturas y notas de credito.

## Pendiente operativo
- Renovacion de vencimientos.
- Pagos a cuenta mas complejos.
- Integracion bancaria real.
