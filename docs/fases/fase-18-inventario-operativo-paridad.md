# Fase 18 - Inventario operativo

## Objetivo
Completar la capa operativa avanzada de Inventario para acercarla a Tango Stock.

## Alcance implementado
- Ensambles y desensambles.
- Cierres de periodo de inventario.
- Revalorizaciones de costo.
- Parametros de stock negativo por tipo de operacion.
- Exposicion web y API de estas operaciones.

## Validacion tecnica
- Migracion de tablas operativas.
- Integracion con costeo por capas y movimientos.

## Pendiente operativo
- Bloqueo duro por periodo cerrado en todos los flujos.
- Politicas avanzadas de picking/logistica interna.
