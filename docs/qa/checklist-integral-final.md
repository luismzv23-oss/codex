# Checklist QA integral final

## Compras
- crear proveedor
- crear orden de compra
- confirmar orden
- recepcionar total o parcial
- verificar impacto en stock
- registrar cuenta a pagar
- registrar pago

## Inventario
- crear deposito
- crear producto
- registrar ingreso
- registrar egreso
- registrar ajuste
- verificar trazabilidad
- verificar kardex valorizado

## Ventas
- crear cliente
- crear venta borrador
- confirmar venta
- verificar stock descontado
- cancelar o devolver
- verificar receivable y comision

## Cobranzas
- registrar recibo
- aplicar a comprobante pendiente
- verificar baja de saldo
- verificar movimiento de caja

## Caja
- abrir caja
- vender en POS/Kiosco
- registrar ingreso/egreso manual
- cerrar caja
- validar arqueo

## Fiscal
- completar settings ARCA
- validar certificado y clave
- ejecutar prueba fiscal
- autorizar comprobante
- consultar comprobante
- revisar bitacora ARCA
